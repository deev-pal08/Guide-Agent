# About Me

Copy this file to `AboutMe.md` and fill in your profile. The agent uses this for cross-class awareness — knowing your background helps it skip basics you already know.

Suggested sections:
- Background (years in security, role)
- Bug classes you've already mastered (e.g. DOM XSS, CORS, SSRF)
- Stack expertise (e.g. JS, Python, Go, AWS)
- Goals (optional — guide is mostly goal-agnostic by design)
